Давайте воспользуемся политикой безопасности контента и воспроизведем поведение первого уровня игры, добавив в ответ веб-сервера нужные хедеры, которые запретят выполнение встроенных скриптов (тех, которые мы помещали внутри хедера и которые запускались уже после загрузки всей страницы).
Для верстки можно воспользоваться следующим шаблоном:

```html
<!DOCTYPE html>
<html lang="en">
<head>
   <meta charset="UTF-8">
   <title>Title</title>
</head>
<body>
 {user_input}
</body>
</html>
```
